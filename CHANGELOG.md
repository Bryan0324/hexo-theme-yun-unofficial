### Theme Modes
- **Light Mode**: Clean, bright theme for daytime
- **Dark Mode**: Easy on the eyes for nighttime
- **Auto Mode**: Intelligent time-based switching
  - 07:00-17:00: Light mode
  - 17:00-20:00: Sunset mode
  - 20:00-07:00: Dark mode
- **Sunset Mode**: Unique warm evening palette with orange/pink tones

### Components
- `Layout`: Main layout wrapper with theme integration
- `ThemeModeToggle`: Floating button for mode switching
- `Banner`: Hero section with title, clouds, and scroll indicator
- `Avatar`: Profile image with rounded style and status
- `SocialLinks`: Flexible social media link display
- `PostCard`: Blog post preview with rich metadata
- `Reward`: Donation/reward section with multiple payment methods
- `CreativeCommons`: License information display
- `Fireworks`: Interactive click-to-explode effect
- `BackToTop`: Smooth scroll to top button
- `Footer`: Enhanced footer with branding and links
